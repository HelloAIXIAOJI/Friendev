# Friendev

## 💬 输入快捷键

```
Enter         →  发送消息
Alt+Enter     →  换行（多行编辑）
Ctrl+Enter    →  换行（备选）
Shift+Enter   →  AI 优化提示词（自动补全细节、添加上下文）
! 开头        →  AI 优化提示词（例如：!写个排序算法）
```

## 🎯 基本操作

```
ESC           →  停止 AI 生成
Ctrl+C (x2)   →  退出 Friendev
Ctrl+D        →  退出 Friendev（空行时）
↑ / ↓         →  浏览历史命令
```

## ✂️ 编辑快捷键

```
Ctrl+A        →  行首
Ctrl+E        →  行尾
Ctrl+U        →  删除到行首
Ctrl+K        →  删除到行尾
Ctrl+W        →  删除前一个单词
Alt+← / →     →  单词跳转
```
